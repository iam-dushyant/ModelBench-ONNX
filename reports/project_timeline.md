# Project Timeline v1

The below information is created for reference to plan, manage and track project timelines.

## Project Objective

This projects forms a part of the 6-month skill development milestone in Machine Learning, Deep Learning, Transformers, MLOps and Production-level ML systems.
The project is considered successful when -
* Exporting a trained model to ONNX and running inference using ONNX runtime.
* Documenting learning for each week, and hence tracking progress by the below project plan.
* GitHub repository should contain all elements of the project, including source code, technical documentation, research sources, and a final conference paper style project report.
